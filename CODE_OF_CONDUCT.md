# Code of Conduct / Кодекс поведения

## Scope / Область

This repository is a public template for AI-assistant bootstrap files. Discussion should stay technical, evidence-based, and focused on improving the template.

Этот репозиторий — публичный шаблон для bootstrap-файлов AI-ассистентов. Обсуждения должны оставаться техническими, проверяемыми и сфокусированными на улучшении шаблона.

## Expected Behavior / Ожидаемое поведение

- Be respectful and direct.
- Critique ideas, code, and documentation, not people.
- Keep reports reproducible: include versions, commands, paths, and expected behavior.
- Do not publish secrets, tokens, private logs, or personal data in issues, pull requests, or discussions.

- Общайтесь уважительно и прямо.
- Критикуйте идеи, код и документацию, а не людей.
- Делайте сообщения воспроизводимыми: указывайте версии, команды, пути и ожидаемое поведение.
- Не публикуйте секреты, токены, приватные логи и персональные данные в issues, pull requests или discussions.

## Unacceptable Behavior / Недопустимое поведение

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Deliberately derailing technical work.
- Sharing credentials or asking others to expose credentials.

- Оскорбления, угрозы, травля или дискриминационные формулировки.
- Публикация приватной информации без разрешения.
- Намеренный срыв технической работы.
- Публикация учётных данных или просьбы раскрыть чужие учётные данные.

## Reporting / Как сообщить

For ordinary project issues, use GitHub Issues. For sensitive reports, do not include secrets in public text; describe the affected area and ask for a private handoff channel.

Для обычных вопросов используйте GitHub Issues. Для чувствительных сообщений не вставляйте секреты в публичный текст; опишите затронутую область и запросите приватный канал передачи деталей.
